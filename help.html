Benutzungsanleitung

Willkommen beim Sitzplangenerator!

Mit diesem Tool können Sie einfach und schnell einen Sitzplan für Ihre Klasse erstellen. Gehen Sie Schritt für Schritt vor:

  1. Geben Sie oben die gewünschte Anzahl der Sitzplätze ein und klicken Sie auf "Sitzplätze neu zeichnen".
  2. Tragen Sie die (Nachnamen,) Vornamen der Schüler/innen semikolon-getrennt in das Namensfeld ein.
    Syntax: Nachname, Vorname; nur_Nachname,; nur_Vorname.
    Beispiel: Albrecht, Anna; Bauer,; Clara.
  3. Falls Sie die Sitzordnung anpassen möchten, können Sie Sitzplätze per Drag & Drop frei verschieben.
    Tipp: Wenn Sie einen manuellen Sitzplan erstellen möchten, lassen Sie zuerst die Namen mit "Namen übertragen" auf die Plätze setzen und verschieben Sie danach die Sitzplätze. So behalten Sie leichter den Überblick, da die ursprüngliche Reihenfolge nach dem Verschieben sonst nicht mehr sichtbar ist.
  4. Weisen Sie nun die Namen den Plätzen zu:
    • "Namen übertragen": ordnet die Namen in der Reihenfolge der Eingabe zu.
    • "Sitzplätze auslosen": verteilt die Namen zufällig auf die Plätze.
  5. Wenn alles passt, können Sie Ihre Eingaben mit "Sitzplätze speichern" und "Namen speichern" dauerhaft im Browser sichern. So gehen sie beim nächsten Öffnen nicht verloren.
  6. Über den Button "PDF Export" können Sie den fertigen Sitzplan als PDF-Datei speichern oder ausdrucken.

Viel Erfolg beim Erstellen Ihres Sitzplans!

Hinweis: Verbesserungsvorschläge können gerne an die im Impressum hinterlegte E-Mailadresse gesendet werden.
